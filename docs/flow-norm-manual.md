# Методичка: нормо-час по 2D-чертежу (режим «Поток»)

Справочный документ для модуля `flow_norm_hours.py`. Расчёт ведётся **без 3D-модели** — только по данным распознавания чертежа (канал email) и параметрам заготовки/станка/партии.

## 1. Подготовка

- **Материал** (штамп) — базовая скорость резания.
- **Ra / допуски** — число чистовых проходов.
- **Габариты** — масса, тип станка, вылет.

## 2. Структура времени

Tшт-к = Tо + Tв + Tтех + Tорг + Tпл + Tпз/n

Упрощение: **Tшт ≈ Tо × Kвсп**.

## 3. Машинное время Tо

Токарное: Tо = L×i / (n×f), n = 1000×Vc / (π×D).

Фрезерное: Tо = L×i / Fмин.

## 4. Kвсп (ориентиры в коде)

| Тип | Kвсп |
|-----|------|
| Универсальные токарный/фрезерный | 1.4–1.7 (в коде 1.55) |
| ЧПУ | 1.2–1.3 (в коде 1.25) |
| Сложные корпуса | 1.8–2.0 (в коде 1.9) |

## 5. Доп. факторы на чертеже

- Малые внутренние радиусы карманов.
- Глубокие отверстия H > 3D.
- Канавки, подрезка торца.

## 6. Тпз

- Универсальные: 20–40 мин/партию.
- ЧПУ: 60–120 мин/партию.

Коэффициенты и эвристики расширяются в `flow_norm_hours.compute_norm_hours()`.
